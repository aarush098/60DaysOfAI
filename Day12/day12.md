- **Python for data analysis** (pandas, numpy) — in progress, no applied project
- **Formal Statistics** — coursework is "statistics-adjacent," not a dedicated stats/probability project or course
- **ETL concepts** — not present at all
- **Machine Learning fundamentals** — listed only as "AI/ML Basics," no depth shown
- **Real business/enterprise dataset experience** — all current projects are self-built web apps, not analysis of external/structured datasets

### Recommended Learning Roadmap (Next 60–90 Days)
1. **Weeks 1–2:** Complete a structured Excel course (pivot tables, VLOOKUP/XLOOKUP, data cleaning) — build one mini project (e.g., budget or sales tracker)
2. **Weeks 2–4:** Learn SQL for analytics specifically (window functions, joins, aggregations) using a free dataset (Kaggle, or Indian government open data)
3. **Weeks 3–5:** Build one Power BI or Tableau dashboard from a public dataset (e.g., sales, COVID data, sports stats) — this single project would be the biggest resume upgrade
4. **Weeks 5–7:** Python for data analysis — pandas + matplotlib/seaborn, applied to the same dataset used for the dashboard, to show an end-to-end pipeline (clean → analyze → visualize)
5. **Weeks 7–8:** Basic statistics refresher (descriptive stats, correlation, hypothesis testing basics) — many free resources (Khan Academy, freeCodeCamp)
6. **Weeks 8–10:** Learn ETL basics conceptually (what it is, simple tools like Alteryx or even Python scripts that simulate extract-transform-load)
7. **Ongoing:** Publish 1–2 of these projects on GitHub with a clear README, and link them in your resume/LinkedIn — this closes your biggest current gap: no analysis project outside of web development

## Job Search Strategy

Java Developer Job Search Strategy 

Reality check: Your resume is currently Data Analyst-focused with only one Java mention (HackerRank). You'll need a Java-specific resume + at least one Java project to be competitive.
1. Daily Plan

Apply to 5–7 tailored roles/day
30 min Java/DSA practice
3–5 LinkedIn connects with Java devs/recruiters

2. Best Portals
LinkedIn, Naukri, Instahyre, Internshala, Wellfound, HackerRank job board, + company career pages (TCS/Infosys/Wipro off-campus drives)
3. Networking
Alumni on LinkedIn > cold applying. Message 5 Java devs/day, post weekly about your Java/DSA progress, use your placement cell.
4. Resume Tips

Build a separate Java-focused resume
Move Java to top of skills, expand beyond "HackerRank"
Rewrite summary around backend/OOP, not data analysis
Add 1 Java project (critical gap right now)

5. Interview Prep

Wk 1–3: Core Java (OOP, Collections, Exceptions)
Wk 2–5: DSA in Java (2–3 problems/day)
Wk 4–6: Build 1 Java project (console app or Spring Boot API)
Wk 5–7: Spring Boot + JDBC basics
Ongoing: Mock interviews, prep your "why the pivot" answer
